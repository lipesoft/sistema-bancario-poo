# Sistema Bancário em Python (POO)

## Sobre o Projeto

Este projeto é uma implementação de um sistema bancário simples utilizando **Programação Orientada a Objetos (POO)** em Python. O código segue rigorosamente o diagrama UML fornecido, demonstrando os principais conceitos de POO:

- **Abstração**
- **Encapsulamento**
- **Herança**
- **Polimorfismo**
- **Classes Abstratas**
