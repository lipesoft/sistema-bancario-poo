# Sistema Bancário em Python (POO)

### Sobre o Projeto

Este projeto é uma implementação de um sistema bancário simples utilizando **Programação Orientada a Objetos (POO)** em Python. O código segue rigorosamente o diagrama UML fornecido, demonstrando os principais conceitos de POO:

- **Abstração**
- **Encapsulamento**
- **Herança**
- **Polimorfismo**
- **Classes Abstratas**

# Como Executar o Projeto

### Pré-requisitos
Python 3.8 ou superior instalado

Git (opcional, para clonar)

### Passo a Passo
1. Clone o repositório (ou baixe os arquivos)

git clone https://github.com/lipesoft/sistema-bancario-poo.git

2. Crie e ative um ambiente virtual (recomendado)
Windows (PowerShell/cmd):

python -m venv .venv
.venv\Scripts\activate

3. Instale as dependências (opcional, apenas para testes)

pip install -r requirements.txt

4. Execute o programa

Opção 2 - Executando como módulo (da pasta raiz):

bash
python -m src.main
